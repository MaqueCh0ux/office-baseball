# Office Baseball

Office Baseball is a tiny baseball franchise sim built inside an Excel spreadsheet.

Play games one plate appearance at a time, generate box scores, track player stats, crown season champions, manage service time, replace players, and reset the franchise when the baseball gods demand blood.

## Current Version

v0.01

## File Size

Approximately 144 KB

## Requirements

- Microsoft Excel
- Macros enabled

## Important

This workbook uses VBA macros. Excel will likely ask you to enable macros before playing.

For accurate stat tracking, generate a new box score after each finished game.

## What is currently included

- New Game button
- Next PA button
- Box score generation
- Hitter and pitcher career stat tracking
- Team records
- All-time team records
- Season length setting
- Championship tracking
- Service time
- Career-limit warnings
- New Player rerolls
- Attribute-driven batter/pitcher bonuses
- Full franchise reset

## Not currently included

OB WAR is an approximation.

Stolen bases, defense, errors, and other secondary baseball events are not currently tracked or directly used in gameplay. Some attributes may exist as future expansion hooks.
